Age Calculator Web App

I built a responsive Age Calculator using HTML, CSS, and JavaScript. Users can enter their date of birth and instantly calculate their age in years, months, and days.

Features:
• Modern and responsive design
• Accurate age calculation
• User-friendly interface
• Dynamic date validation
• Clean and organised code

Technologies Used:
• HTML5
• CSS3
• JavaScript

This project helped me strengthen my front-end development skills, including  date calculations and responsive web design.
